# 🏐 SUPER SPIKE

**웹 데스크톱 3 vs 3 아케이드 배구** 게임입니다. 코트를 비스듬히 위에서 내려다보는 **3/4 탑다운 원근 뷰**로, 공은 x·y(높이)·z(깊이) 3D 물리로 움직입니다.
Vite + TypeScript + HTML5 Canvas로 만들어졌으며, 외부 에셋 없이 캔버스로 그린 애니풍 캐릭터와 절차적(Web Audio) 사운드로 동작합니다.

## 🎮 플레이 방법

| 조작 | 키 |
| --- | --- |
| 코트 이동 (4방향) | `W`/`A`/`S`/`D` 또는 방향키 (`W`/`↑` = 네트 쪽 전진) |
| 점프 | `Space` |
| 스윙(리시브·토스·스파이크) | `J` / `K` / `마우스 클릭` |
| 블록 | 네트 근처에서 점프 후 `J` **유지** |
| 일시정지 | `Esc` / `P` |

### 팀 플레이 — 공 근처 아군 자동 조종
- 우리 팀은 **3명**(후위·중위·전위). 플레이어는 **공에 가장 가까운 아군을 자동으로 조종**합니다(머리 위 🟢초록 화살표·발밑 링으로 표시).
- 점프/스윙 중에는 전환이 잠겨(락) 스파이크가 중간에 끊기지 않습니다.

### 핵심 재미 — 리시브 → 토스 → 스파이크 콤보
- **타격 방향은 터치 순서로 결정됩니다.**
  - 1·2번째 터치(리시브/토스): 공이 **우리 코트로 높이** 떠서 다음 아군이 공격할 수 있게 세팅됩니다.
  - 네트 앞에서 **점프 스파이크**하거나 3번째 터치는 **상대 코트로** 넘어갑니다.
- 즉, 후위가 리시브 → 자동 전환된 세터가 토스 → 전위로 전환해 **점프 타이밍 스파이크**로 마무리하는 흐름이 자연스럽게 나옵니다.
- 네트 앞 점프 후 **타이밍 링**이 가장 작아지는(점프 정점) 순간 `J` → **PERFECT 스파이크**(강타 + 화면 흔들림·슬로우모·파티클).
- 상대 스파이크에 맞춰 네트에서 점프+`J`유지 → **블록**.
- 한 팀은 최대 **3번**까지만 터치할 수 있습니다(4번째는 실점).

### 규칙
- 랠리 스코어링. 한 세트 **7점**(듀스 시 2점 차), **3판 2선승**.
- 랠리가 길어질수록 공이 빨라지고 관중이 뜨거워집니다.

## 🛠 개발

```bash
npm install
npm run dev        # 개발 서버 (http://localhost:5173)
npm run build      # 프로덕션 빌드 → dist/
npm run preview    # 빌드 결과 미리보기
npm run typecheck  # 타입 검사
```

### 헤드리스 시뮬레이션 (로직 검증)
봇 vs AI 자동 플레이로 물리·득점·세트 전환을 크래시 없이 검증합니다.

```bash
node_modules/.bin/esbuild sim/headless.ts --bundle --format=esm --platform=node --outfile=/tmp/headless.mjs
node /tmp/headless.mjs
```

## 🚀 배포 (Vercel)

정적 SPA로 그대로 배포됩니다.

```bash
npm i -g vercel
vercel          # 프리뷰
vercel --prod   # 프로덕션
```

Vercel이 Vite를 자동 인식하며(`vercel.json` 포함), 빌드 산출물은 `dist/` 입니다.

## 📁 구조

```
src/
  main.ts            앱 셸 · 씬 매니저(타이틀/선택/매치/일시정지)
  engine/
    math.ts          벡터·보간·PRNG·이징
    input.ts         키보드/포인터 입력(엣지 트리거)
    audio.ts         절차적 SFX + 배경음 (Web Audio)
    particles.ts     파티클 시스템
  game/
    config.ts        3D 코트 치수·물리 상수·포메이션·팔레트·난이도
    camera.ts        원근 투영(월드 x·y·z → 화면 좌표)
    ball.ts          3D 공 물리(네트·바닥 충돌)·궤적·착지 예측
    fighter.ts       캐릭터 3D 물리(코트 이동·점프)·스윙·애니 포즈
    ai.ts            AI 컨트롤러(반응 지연·조준 오차·스파이크/블록)
    effects.ts       스크린 셰이크·히트스톱·슬로우모·충격파·스피드라인·팝업
    world.ts         시뮬레이션(3D 타격 판정·득점·서브·세트 흐름)
    render.ts        원근 코트·네트·깊이정렬 캐릭터·공·HUD 렌더링
    menus.ts         타이틀·선택·일시정지·결과 화면
sim/
  headless.ts        헤드리스 자동 플레이 검증 하네스
```
